2. HTML Structure

Maverick consist of 4 different sections.

Maverick utilizes Bootstrap's 12 column 1170px Grid System. What that means for you is, you can fit in more content then normal 960 grid websites. Our website is divided into four sections. Here are the sections broken down:

  • Top Hero · Includes the navigation bar, the phones, and the text, inside the hero (top portion of the website). This is what the customers will first see, so we tried to make it as appealing as possible.
  • Tour · Includes all the elements from the Tour section, such as the Introduction, Features, and the Video. In here, you will show off all the cool features your product has. We have provided you with three essential sections that you can start with to share the features. Note for the Features section on the PSDs: I have rasterized the icons in the background for the demo sake, due to the fact that I cannot apply Clipping Mask for a folder. So if you wish to edit the icons, I have provided the vector folder with all the icons. Simply unhide it and start editing!
  • Testimonial · Section with all the customer reviews.
  • Footer/Buy Now · Has the buy now function, as well as the footer credits. I have also created the App Store and the Play Store badge, so feel free to use them however you want.

8. Support & FAQs

Q: How do I change my logo?
A: You can change the logo by locating 
#logo { ... }
, and renaming logo.png to your own! Make sure it's still named logo.png.

Q: How do I move the pointers in Testimonials?
A: You can change the locations of the pointers by changing the top and left of 
.location#t_purple a i, #testimony_1 h5 span { ... }
located in main.css. Note: Make sure you edit them in the section commented: "Testimonial Maps"!

Q: How do I change the thumbnail on the Laptop screen?
A: You will have to put up your own thumbnail image. Make sure the size is the same as the one in the download: (Size: 569x356)

Q: Why is the video not showing up?
A: The video player is not showing up, because it requires a server in order to connect to YouTube. Viewing it from your localhost is not possible. But do not worry! When you upload the website to the web, the video will show up.

Q: How do I add in more images to the slider?
A: You can add more images to the slider by dropping the image into the slider directory located in the image folder. Then locate 
<li><img src="assets/img/slides/3.jpg" alt=""></li>
and just add in another line and changing 3.jpg to whatever the file you placed and it's extension is. (Size: 279x489)

Q: How do I make an image fit in a responsive layout?
A:Simple! Locate the image you wish to make it scale with the grid, and type in 
class="img-responsive"
. So the whole code will look like this: 
<img src="YOURPICTUREHERE.PNG" class="img-responsive">


Q: Where do I declare custom jQuery functions?
A:You can declare custom jQuery functions in assets/js/custom.js.

Q: Where do I add custom CSS styling?
A:You can add custom CSS styling in assets/css/custom.css.

Q: How do I change the image preview in the Introduction?
A:You can change the image on the Phone by going to the "slides" folder in the "img" folder, and replacing the current feature.jpg with your new one. Make sure it's still called feature.jpg.
